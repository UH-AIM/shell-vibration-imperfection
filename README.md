# Shell Acoustic Imperfection

This repository contains all digital materials associated with the manuscript  
**“Shell Acoustic Imperfection”**.

The repository is intended to support transparency, reproducibility, and data availability for the results presented in the main text and Supplementary Information (SI).

## Repository Contents

The repository includes:

- **Data**
  - Raw and processed data used to generate figures in the main manuscript and the Supplementary Information.
  - Data files are organized by figure or analysis where applicable.

- **Scripts**
  - Scripts used for data processing, analysis, and figure generation.
  - These scripts reproduce the plots and quantitative results reported in the manuscript and SI.

- **Manuscript Files**
  - LaTeX source files for the main manuscript.
  - LaTeX source files for the Supplementary Information.

## Notes

- All figures in the manuscript and SI can be reproduced using the provided data and scripts.
- File and folder names are chosen to correspond directly to figures or sections in the paper.
- This repository will be made publicly accessible upon acceptance of the manuscript.
